# ESP32 Smart Car with L298N

Advanced ESP32 smart car project supporting:

- WiFi Control
- Bluetooth Control
- MQTT Control
- Web Server Interface
- PWM Speed Control
- Android App Support

---

# Hardware Components

- ESP32 Dev Module
- L298N Motor Driver
- 4x DC Motors
- 12V Battery
- LEDs
- Chassis Robot Car

# Pin Connections

| L298N | ESP32 |
|------|------|
| ENA | GPIO21 |
| IN1 | GPIO4 |
| IN2 | GPIO23 |
| IN3 | GPIO18 |
| IN4 | GPIO19 |
| ENB | GPIO22 |
